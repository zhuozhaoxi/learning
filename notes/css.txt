头部fixed固定，滚动到一定程度下滑出现
js监控，符合条件时 添加class
class{
	animation: topScrollDown .5s ease-in-out;
}

@keyfrmas topScrollDown {
	0%{
		top: -50px;	// 这里是头部的高度
	}
	to{
		top: 0px;
	}
}


利用after清除浮动
.clearfix:after {content:"."; display:block; height:0; visibility:hidden; clear:both; }
.clearfix { *zoom:1; }
